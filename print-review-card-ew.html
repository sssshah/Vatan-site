How to use this file:
1. Open in Chrome or Safari
2. Wait a second for the QR codes to generate (they appear automatically)
3. File → Print → set paper to Letter, margins to None or Minimum
4. You'll get 4 cards per page — cut along the dashed lines
5. Laminate or print on cardstock for table use

QR code links to: East Windsor Google Reviews
To reprint JC cards: open print-review-card-jc.html
